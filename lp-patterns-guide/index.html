OVERVIEW LP構成パターンの分類体系

LPの構成パターンは大きく2つのカテゴリに分かれる。
「正式なフレームワーク」と「実務上の分類」は性質が異なるため、本ガイドでは明確に区別して扱う。

SECTION 1: 正式なフレームワーク

マーケティング研究者・実務家が体系化し、名前をつけて提唱したもの。
LP全体の「骨格」を決める設計図。

BEAF 実績×論理で証明する
PASONA 不安を煽って行動させる
新PASONA 共感から寄り添って解決する
QUEST 教育→納得→行動の段階設計

SECTION 2: 実務上の分類パターン

勝ちLP 52件の実データ分析から抽出した、よく見られる「表現手法・要素」の分類。
メインのフレームワークと組み合わせて使うサブ要素。

ストーリー型 開発秘話・体験談で感情を動かす
比較・切替型 Before/After・他社比較で差別化
教育・啓蒙型 メカニズム解説で判断基準を作る
UGC/口コミ型 社会的証明・ランキングで信頼構築

正式 vs 実務の違い

正式なフレームワークは「LP全体の骨格」を決める。1つのLPに対して主軸は1つ。
実務上の分類は「表現の味付け」。主軸フレームワークの上に複数を重ねて使うのが一般的。
実際の勝ちLPでは 82% が「メインフレームワーク + 1〜4個のサブ要素」を組み合わせている。

特徴

提唱者
神田昌典（1999年）。著書『禁断のセールスコピーライティング』で体系化。日本のDRM（ダイレクト・レスポンス・マーケティング）の父。
核心
「A = Agitation（煽り）」がキモ。問題を放置すると「こんなに大変なことになる」と恐怖・不安を拡大し、解決への渇望を生む。
現代での扱い
煽りが強すぎると薬機法・景表法に抵触するリスクあり。現代の通販LPでは「新PASONA」に進化。旧PASONAを純粋に使うLPはほぼ見られない。

旧PASONA → 新PASONAの進化

A: Agitation
旧: 不安を煽る（恐怖訴求）
新: Affinity（親近感） — 共感・寄り添い
SO → S+O 分離
旧: SOlution（解決策）1ステップ
新: Solution + Offer に分離。解決策の説明と具体的な提案を分ける
設計思想の転換
「恐怖で動かす」→「共感で動かす」。ネガティブ→ポジティブへのパラダイムシフト

勝ちLP 52件の中に「旧PASONA」を純粋に使っているLPは0件。
すべて「新PASONA」に進化した形で採用されている。フレームワークの歴史・背景として理解しておくべき知識。

SECTION 2

実務上の分類パターン

勝ちLP 52件の実データから抽出した、頻出する「表現手法・要素」の分類。
これらは独立したフレームワークではなく、メインフレームワーク（BEAF / 新PASONA）に重ねて使うサブ要素。

学術的に名前がついているものではなく、
実際の勝ちLPを観察して「よく使われているパターン」を体系化したもの。

ストーリー型 比較・切替型 教育・啓蒙型 UGC/口コミ型

PRACTICAL 01 ストーリー型 7件 / 14%

開発者の想い・開発秘話・体験談をストーリーテリングで語り、感情的な共感と信頼を獲得する手法。
「なぜこの商品が生まれたのか」という物語が、商品に唯一無二の価値を与える。

よく使われる要素

開発秘話
「◯年かけて◯◯回の試作を経て完成」「諦めかけた時に出会った◯◯」など、商品誕生の物語。
創業者の想い
「自分自身が悩んでいた」「家族のために作った」など、パーソナルな動機。共感の起点。
監修者ストーリー
皮膚科医・専門家の「なぜ監修を引き受けたか」。権威性にストーリーを重ね、深い信頼を作る。
体験談型
ユーザーの「出会い→使用→変化」を時系列で描写。主人公がユーザー自身になるパターン。

効果的な使い方

配置のコツ

LPの中盤〜後半に配置するのが一般的。前半で商品メリットや悩み共感を展開した後、「なぜこの商品が信頼できるか」の文脈でストーリーを入れる。

組み合わせ相性
新PASONA との相性が特に良い。共感（A）→ 解決策（S）の流れの中に開発ストーリーを挟むと、解決策の説得力が増す。

注意点

ストーリーだけでは売れない。あくまで「信頼構築」の要素であり、メインフレームワークの論理構造が土台として必要。出現率14%が示す通り、全LPに必要なわけではない。

PRACTICAL 02 比較・切替型 20件 / 41%

Before/After、他社比較、従来品との違いなどを視覚的に対比させ、商品の優位性を直感的に伝える手法。
「今使っているもの」から「この商品」へのスイッチングを促す。

よく使われる要素

Before/After
使用前→使用後の変化を写真・イラストで視覚化。最も直感的で強力な訴求手法。薬機法の制約に注意。
他社比較表
成分量・価格・特徴を一覧表で比較。「◯」「×」で自社の優位性を視覚的に示す。
従来品との違い
「従来の◯◯は△△だった。この商品は◯◯」。同カテゴリ内での世代交代・進化を訴求。
対比構造
テキスト・デザインでの対比。「◯◯なあなた」vs「△△なあなた」で選択を迫る。

メイン別の出現率

BEAF +
42% (8/19件)
新PASONA +
41% (12/29件)

BEAF・新PASONAどちらとも約40%の出現率。メインフレームワークを問わず汎用的に使われている。

配置のコツ

BEAFの「A（Advantage）」セクションと特に相性が良い。競合比較表を置くことでAdvantageの説得力が格段に上がる。新PASONAでは「S（Solution）」の中で「従来品との違い」として活用。

PRACTICAL 03 教育・啓蒙型 16件 / 33%

悩みの原因メカニズム、成分の仕組み、技術の解説など、専門知識を提供して「納得」させる手法。
QUESTフレームワークの「E = Educate」を、サブ要素として他のフレームワークに組み込んだ形。

よく使われる要素

原因メカニズム
「なぜ肌が乾燥するのか」「シミの原因は◯◯層の△△にある」。悩みの根本原因を図解で解説し、解決策への導線を作る。
成分・技術解説
「独自のナノ浸透技術」「ヒト幹細胞培養液とは」。成分や技術の仕組みを科学的に説明し、効果への期待を論理的に裏付ける。
処方・設計思想
「7つの無添加」「皮膚科医監修の処方設計」。なぜこの配合なのか、なぜこの設計なのかを説明。
業界の新常識
「実は◯◯は逆効果だった」。既存の常識を覆す新事実を提示し、ユーザーの判断基準を書き換える。

メイン別の出現率

BEAF +
26% (5/19件)
新PASONA +
38% (11/29件)

新PASONAとの組み合わせが多い。「P（問題）→ A（共感）→ 教育（なぜ？）→ S（解決策）」という流れで、教育がSolutionの説得力を高める。

配置のコツ

新PASONAなら A（共感）→ 教育セクション → S（解決策） の間に挟む。「あなたの悩みの原因はこれ」→「だからこの成分が効く」の論理展開。
BEAFなら A（Advantage） 内で「なぜ他と違うのか」の技術的裏付けとして使う。

PRACTICAL 04 UGC/口コミ型 25件 / 51%

SNS投稿、@cosmeレビュー、雑誌掲載、ランキング受賞、累計販売数など「第三者の評価」で信頼を構築する手法。
勝ちLPの過半数（51%）に含まれる、最も汎用的なサブ要素。

よく使われる要素

SNS口コミ
Instagram投稿風のデザインで「リアルな声」を演出。ハッシュタグ付きの投稿形式が多い。信憑性と親近感を両立。
ランキング受賞
「@cosme第1位」「楽天ランキング1位」。複数のランキングを横並びに配置し、受賞実績の量で圧倒する。
メディア掲載
「VOCE」「MAQUIA」等の美容雑誌ロゴを並べる。権威あるメディアに取り上げられた実績で信頼性を担保。
累計数・実績
「累計◯◯万個突破」「シリーズ累計◯億円」。大きな数字を視覚的にインパクトあるデザインで表示。

FV直下: 累計数・ランキングバッジを帯状に配置。ファーストビューの信頼性を底上げ。

CTA直前: 「これだけの人が選んでいます」で最後の一押し。

DECISION FLOW パターンの使い分けガイド

ターゲット層と商品特性から最適なパターンを選ぶフローチャート。

Step 1: メインフレームワークを決める

BEAF を選ぶ
以下に当てはまる場合:
・ランキング受賞、メディア掲載など実績が豊富
・累計販売数や満足度の数字が強い
・ターゲットはすでに買う気がある顕在層
・「どれにしよう」と比較検討中のユーザー
新PASONA を選ぶ
以下に当てはまる場合:
・ユーザーに明確な悩み・痛みがある
・まだ解決策を決めていない準顕在層
・共感・寄り添いが効くカテゴリ（美容・健康等）
・実績よりもストーリーや想いで差別化する商品

Step 2: サブ要素を選ぶ（複数可）

UGC/口コミ
口コミ・SNS投稿・ランキング受賞がある → 迷ったら入れる。最も汎用的。
特にBEAFとの相性◎（68%が採用）
比較・切替
競合との明確な差別化ポイントがある → 比較表・B/Aで視覚化。
BEAF・新PASONAどちらでも40%超が採用
教育・啓蒙
独自技術・成分・メカニズムの説明が必要 → 図解・断面図で解説。
新PASONAとの相性◎（38%が採用）
ストーリー
開発者の想い・開発秘話がある → ブランド信頼構築に効く。
出現率14%。全LPに必須ではないが、あると深みが出る

迷ったら「新PASONA + UGC + 比較」

判断に迷う場合は、新PASONA + UGC/口コミ型 + 比較・切替型 の組み合わせが最も安全。
実際の勝ちLPでも4件（TOP3タイ）がこの組み合わせを採用しており、バランスの取れた構成。
「共感で引き込み → 口コミで安心させ → 比較で納得させる」の三段構えが基本。
